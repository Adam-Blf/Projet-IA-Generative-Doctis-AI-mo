# Contribuer à Doctis-AI-mo

Merci de votre intérêt ! Voici comment contribuer :

1. **Forkez** le projet.
2. Créez une **branche** (`git checkout -b feature/MonIdee`).
3. **Commettez** vos changements en suivant les standards (Conventional Commits).
4. **Pushez** sur votre Fork.
5. Ouvrez une **Pull Request**.

## Règles

* Variables en Anglais.
* Commentaires en Français.
* Pas de BDD locale (Utilisez le Mock Data pattern).
