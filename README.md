# DoctisAImo (v3.0-KAGGLE-MAPS)

DoctisAImo is an advanced AI emergency triage assistant designed to analyze patient symptoms and provide safety assessments in multiple European languages.

## Overview

This project uses Generative AI to provide:

- **Triage Urgency**: Rapid safety assessment using Kaggle-style data-driven logic.
- **Second Opinion**: Detailed risk analysis based on disease probability distributions.
- **Action Plan**: Immediate checklist for emergencies.
- **Input Enrichment**: Semantic keyword generation for vector search.

## Configuration

Configuration and prompts are stored in `config/prompts.json`.
